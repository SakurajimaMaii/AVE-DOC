# 身份证验证

> `IDRegex` 主要用于在**本地**验证中国居民身份证的正确性，使用方法如下：

=== "kotlin"

    ```kotlin
    IDRegex.validateIDCardNumber("123456789101112131")
    ```

=== "java"

    ```java
    IDRegex.validateIDCardNumber("123456789101112131");
    ```

下面列举了验证失败的情况：

1️⃣ 身份证号码长度不是15位或18位。

2️⃣ 身份证15位号码不都为数字 ; 18位号码除最后一位外，不都为数字。

3️⃣ 身份证生日无效。

4️⃣ 身份证生日不在有效范围。

5️⃣ 身份证月份无效。

6️⃣ 身份证日期无效。

7️⃣ 身份证地区编码错误。
