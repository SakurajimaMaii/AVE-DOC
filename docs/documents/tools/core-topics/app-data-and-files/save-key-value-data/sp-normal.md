# SpNormal

!!! tip 

    出于安全方面的考虑，我更推荐使用 `SpEncrypted` 。

`SpNormal` 用于创建 `SharedPreferences` 实例。

## 快速开始

[:octicons-tag-24: Version 1.5.0](https://ave.entropy2020.cn/version/tools/#150)

```kotlin
val sp = SpNormal(name).getSharedPreferences()
```
