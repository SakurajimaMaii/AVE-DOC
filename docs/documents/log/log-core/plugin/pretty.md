# Pretty 风格

当我们以字符串形式将 json 进行打印时:

```kotlin
val json = """
    {
            "minInt": -9007199254740991,
                        "maxInt": 9007199254740991,
            "minFloat": -1.7976931348623157e+308,
        "maxFloat": 1.7976931348623157e+308
        }   
""".trimIndent()
logCat.d(json)
```
