# SpNormal

!!! tip 

    出于安全方面的考虑，我更推荐使用 `SpEncrypted` 。

`SpNormal` 用于创建 `SharedPreferences` 实例。

## 快速开始

```kotlin
val sp = SpNormal.getInstance(name).getSharedPreferences()
```
