# SpNormal

!!! tip

    For security reasons, I prefer to use `SpEncrypted`.

`SpNormal` is used to create an instance of encrypted SharedPreferences. 

## Get started

```kotlin
val sp = SpNormal.getInstance(name).getSharedPreferences()
```
